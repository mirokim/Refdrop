Privacy Policy

Last updated: March 2025

1. What RefDrop does

RefDrop is a Chrome extension that lets you send images and video frames from web pages directly into PureRef, a reference image viewer running on your computer. A companion helper app installed locally handles the transfer to PureRef.

2. Data we collect

RefDrop collects and stores the following data locally on your device only, using chrome.storage.local:

  • Send count — a cumulative counter of how many images you have sent (used to enforce the 50-image free tier limit).
  • License status — whether a valid PRO license key has been activated.
  • License key hash — the key you entered, stored locally to persist your PRO activation across browser restarts.
  • Helper status — whether the local helper app is installed (stored as a boolean flag).

None of this data is transmitted to any external server operated by RefDrop.

3. License verification

When you activate a PRO license key, the key is sent to Gumroad's license verification API (api.gumroad.com/v2/licenses/verify) to confirm its validity. This is a one-time request at the moment of activation. After verification, the result is stored locally and no further requests are made to Gumroad.

Gumroad's own privacy policy applies to that request: gumroad.com/privacy

4. Image data

Images you choose to send are transferred directly from your browser to the local helper app via Chrome's Native Messaging API (a local stdio pipe). Images are never uploaded to any server. They exist only in memory during the transfer and are discarded immediately after PureRef receives them.

5. Permissions used

  • storage — stores the send count and license status locally.
  • nativeMessaging — communicates with the locally installed helper app to send images to PureRef.
  • scripting / activeTab — injects the hover button UI into web pages so you can click images.
  • contextMenus — adds a right-click menu option as an alternative to the hover button.
  • <all_urls> — required so the hover button can appear on any website you visit.

6. No tracking, no analytics

RefDrop does not use any analytics service, crash reporter, or telemetry. No usage data is collected or transmitted.

7. Changes to this policy

If the privacy policy changes in a meaningful way, we will update the date above and note what changed.
